## 運営サービス・ライブラリ

### [選挙ウォッチ](https://senkyo.watch)

日本の各政党は常に候補者擁立の準備を進めているものの、既存メディアは選挙期間より前には取りまとめをほとんど公開しない。国政選挙の「いま現在の候補者一覧」を中立的・網羅的に提供する日本唯一の web サービスとして2021年3月に開始。2021年の衆院選と2022年の参院選を経験した。日本の複雑な選挙制度を乗り越えるシンプルな UI による議席予想支援機能も提供し、推定1,000人以上のヘビーユーザーを持つ。

フロントエンドアプリに Next.js (React, TypeScript)、バックエンド構成として Firebase (Firebase Hosting, Cloud Functions, Firestore) を利用している。また、地理的な傾向のわかりやすい可視化に力を入れており、GIS による地理情報の活用も積極的に行っている (QGIS によるマップ作成や、JavaScript 製自作プログラムによるデータ処理など)。

### [jp-local-gov](https://www.npmjs.com/package/jp-local-gov)

都道府県 (JIS X 0401) や全国地方公共団体コード (JIS X 0402)、地域メッシュコード (JIS X 0410) 等、地方行政データを扱う上で便利な関数を取りまとめた npm ライブラリ。
